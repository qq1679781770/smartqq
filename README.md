### SmartQQ:simple version of Java QQ

----------
#### 简介
----
Smartqq是用Java编写的qq的极简易版本，界面还行，功能还行。利用C/S架构，其中smartqq是服务端，smartqqclient是客户端。
#### 配置
----

 - 使用MySQL  
 在smartqq/src/db.properties修改
 jdbc.user jdbc.password jdbc.jdbcUrl
 
 - hibernate
 第一次启动服务时自动创建表
 
 - 先运行服务端，然后运行客户端
